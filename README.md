## How do I install ChromeDriver?

Click the three dots menu → Help → About Google Chrome. Note the version number (e.g., 148.0.7778.168). 
Navigate to the official Chromium ChromeDriver site: https://sites.google.com/chromium.org/driver/. Select the release that matches your Chrome version.
